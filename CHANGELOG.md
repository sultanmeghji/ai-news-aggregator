## Release Notes

### Version 1.0.0
This is the first stable release of the AI News Aggregator. The application provides a complete solution for aggregating AI-related news stories, generating social media content, and automating social media posting.

**Key Features:**
- Automated news aggregation from trusted sources
- AI-powered content summarization using Anthropic Claude
- Social media integration with Social Pilot
- Comprehensive rate limiting and error handling
- Real-time usage monitoring
- Lightweight CSV-based storage

**Getting Started:**
1. Download the latest release
2. Follow the installation guide in README.md
3. Set up your API keys in the .env file
4. Run `npm install && npm run dev`

**Requirements:**
- Node.js 18 or higher
- API keys for News API and Anthropic Claude
- Optional: Social Pilot API key for social media posting

**Breaking Changes:**
- None (initial release)

**Known Issues:**
- None at this time

For detailed installation and usage instructions, see the README.md file.
